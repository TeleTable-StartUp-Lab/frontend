# Running, Build & Testing

## Install

- Install dependencies via the package manager configured in [package.json](../package.json).

## Development

- Start the dev server (CRA default).
- The app runs on the standard local dev port.

## Build

- Create a production build via the scripts in [package.json](../package.json).

## Tests

- Tests are configured via CRA defaults.
- Entry point: [src/App.test.js](../src/App.test.js)
- Test setup: [src/setupTests.js](../src/setupTests.js)

## Environment & Config

- Environment variables are read from CRA conventions.
- Keep secrets out of the frontend.
